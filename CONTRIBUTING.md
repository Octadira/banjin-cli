# Contributing

When contributing to this project, please follow these guidelines.

## Branching Strategy

-   **main**: This branch is for stable releases. Do not push directly to this branch.
-   **develop**: This is the main development branch. All new features and bug fixes should be based on this branch.

## Pull Requests

-   Create a new branch from `develop` for each new feature or bug fix.
-   Once the work is complete, create a pull request to merge the changes back into `develop`.
-   Ensure that your code is well-tested and follows the existing coding style.
